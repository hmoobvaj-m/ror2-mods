# Risk of Rain 2 Mods
This repository contains multiple Risk of Rain 2 mods.
This fork is based on original work by **[MCMrARM]**.

- Original Repository: [MCMrARM/ror2-mods](https://github.com/MCMrARM/ror2-mods)
- Original Author: [MCMrARM](https://github.com/MCMrARM)

This fork currently focuses on updating/improving the original mods of MCMrARM

## Included Mods
### [Chest Item Picker V2](ChestItems/README.md)

Original mod: **Chest Item Picker**  
Fork version: **Chest Item Picker V2**

**Status:** Final Validation

This fork updates the original Chest Item Picker behavior for newer Risk of Rain 2 versions 

### [ItemCounter](ItemCounters/README.md)

Original mod: **ItemCounter**  
Fork version: **ItemCounter**

**Status:** Not yet verified

Adds a counter to track the number of items you have for each tier.

No major fork-specific changes are currently documented.

### [TruePause](TruePause/README.md)

Original mod: **TruePause**  
Fork version: **TruePause**

**Status:** Not yet verified

Allows pausing the game in multiplayer.

## Current Focus
- Updating original mods for current Risk of Rain 2 versions
- Fixing behavior that no longer works currently
- Improving compatibility with vanilla game systems and potentially other mods
- Preserving the original intent of each mod where possible
